# Custom Patches for Youtube
----------------------------

## Introduction

This repository contains userstyles that makes our own Youtube experience better.
It is separated by category, plus a miscellaneous one.

We recommend using it with [Stylus] extension.

## Contents

Description of each userstyle is as follows:

### Hide Youtube BS

Source: `hide-bs.user.less`

Tired of seeing members-only video recommended to you?
Spoiled by AI summary box?
Don't wanna see any paid comments?
Then this userstyle is for you!

You can hide, filter out, or censor out it's content.
Marie Kondo your Youtube experience, the way it's meant to be.

### Restore Old Youtube End Screen Grid

Source: `old-end-screen.user.less`

Youtube hides the old grid view with V key.
This userstyle restores it so you don't have to press v everytime.

### Miscellaneous Patches for Youtube

Source: `misc.user.css`

Our own patches to make Youtube more bearable.
Not guaranteed to work, please use with your own risk.
Some patch may ends up moved to it's own category/file.
Used alongside 'AdashimaaTube as extension/patches for it.

## Possibly Asked Questions

### This Style is Broken!

File an issue describing the thing.
Add screenshot and if possible inspect element that is problematic.

### I Want to Hide X, but Your Style don't Have It

File an issue describing which element should be hidden.
Youtube is doing experiment all the time,
so it might take a while to find the exact element.

### I Want X Replacement

File a PR with your replacement.
If it's good, i might merge it.
Don't be disheartened if your PR got rejected, it's probably ass.

### Why?

Because we're tired of Youtube shennanigans, duh!
At first, we installed extension and userstyles to remedy it.
As time goes on, we write our own little patches.
This is the culmination/archive of those.
