# Custom Patches for Youtube
----------------------------

## Introduction

This repository contains userstyles that makes our own Youtube experience better.
It is separated by category, plus a miscellaneous one.

We recommend using it with [Stylus] extension.

## Contents

Description of each userstyle is as follows:

### Hide Youtube BS

Source: `hide-bs.user.less`

Tired of seeing members-only video recommended to you?
Spoiled by AI summary box?
Don't wanna see any paid comments?
Then this userstyle is for you!

You can hide, filter out, or censor out it's content.
Marie Kondo your Youtube experience, the way it's meant to be.

### Miscellaneous Patches for Youtube

Source: `misc.user.css`

Our own patches to make Youtube more bearable.
Not guaranteed to work, please use with your own risk.
Some patch may ends up moved to it's own category/file.
Used alongside 'AdashimaaTube as extension/patches for it.
